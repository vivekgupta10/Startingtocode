How to make a python (.py) file

You can create and make a python(.py) file using any text editor program.For example windows notepad . so first open notepad write a python code or program you want to run then click on save as, you can save it anywhere you want for example i am saving this on my desktop, when you click on save as you will show two boxes file name and save type as . select all files on save type as and in file name write your disireable name and without a gap write .py which is an extension of python. Then click on save and you first step to make an python file is completed you can edit this file by right clicking on it and click open with notepad.

How to download a python compiler

To execute a python file you need to have a compiler on which you can execute the python file . Open web browser then enter this following url www.python.com then go on download then download the latest python compiler available. NOTE-If you are using windows 7 or earlier version of windows you cannot run the latest python compiler but you can download python 2.7 or earlier version of python which is compatiable with you windows version. After you download the python setup.exe file you can right click on it and click on run as administrator then the setup will continue you can click on accept term and conditions to continue then click on next and then next and then it will take some time and the setup will complete then you click on finish.If you download python 3.0 or latest you can skip to next step but if you download python 2.9 or earlier you have to go on your python pogram folder then you copy the adress then you on Control Panel\System and Security\System then click on advance system settings then click on advance then click on enviromental variables then scroll and select path then click on edit then go to the end then put a semicolon (;) then paste the adress you copy then click on apply then click on ok then exit NOTE-Do not delete or remove any enviromental variable
